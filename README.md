
# 🍲 React Recipe App

A modern and responsive **Recipe Web Application** built using React.js. This project allows users to explore, search, and view detailed recipes with a clean UI and smooth user experience.

---

## 🚀 Features

* 🔍 Search recipes by name
* 📂 Browse recipes by category
* 🧾 View detailed recipe instructions
* ⏱️ Display cooking time and ingredients
* 📱 Fully responsive design
* ⚡ Fast and dynamic UI with React Hooks

---

## 🛠️ Tech Stack

* **Frontend:** React.js
* **Routing:** React Router DOM
* **Styling:** CSS / Tailwind (if used)
* **Icons:** Lucide React
* **API:** MealDB / External Recipe API
* **State Management:** useState, useEffect, custom hooks

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/student-Priyanshu1/my-react-recipe-app.git
```

2. Navigate to project folder

```bash
cd my-recipe-app
```

3. Install dependencies

```bash
npm install
```

4. Start development server

```bash
npm run dev
```
